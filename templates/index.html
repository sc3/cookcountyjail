Welcome to the Cook County Jail inmate tracker

This project aims to provide open data about the inmates in the Cook County Jail. The project is ran by the dedicated contributors of the Supreme Chi-Town Coding Crew. You can join us every Saturday at FreeGeek Chicago. All the data is available through the API and, the source code is hosted on GitHub

The 26thandCalifornia project is a great example of what can be done with these data. The source code for that project is also available on GitHub.
